# 🎬 Projeto Filmes e Paisagens

Meu primeiro projeto em React, desenvolvido para praticar conceitos fundamentais da biblioteca e aprimorar minhas habilidades em desenvolvimento Front-End.

## 🚀 Tecnologias

- React
- JavaScript
- HTML
- CSS
- Vite

## ▶️ Como executar

```bash
npm install
npm run dev
```

Acesse o endereço exibido no terminal (`http://localhost:3000`).

## 📝 Iniciando um novo projeto React

```bash
npm create vite@latest nome-do-projeto -- --template react
cd nome-do-projeto
npm install
npm run dev
```

- Componentes
- JSX
- Estados e Hooks
- Consumo de APIs
